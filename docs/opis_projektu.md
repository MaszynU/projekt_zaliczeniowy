# Opis Projektu

Projekt dotyczy prostej aplikacji webowej pomagającej studentom planować naukę.
Rozwiązuje problem chaosu w zadaniach i braku regularności.
Odbiorcami są studenci w wieku 18–25 lat.
Aplikacja będzie zawierać listę zadań, widok tygodnia oraz podstawowe statystyki postępów.

## Funkcje
- Lista zadań
- Kalendarz tygodniowy
- Proste statystyki

## Cel projektu
Pomóc studentom lepiej planować czas i organizować obowiązki.
